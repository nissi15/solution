# ThinkSphere Platform

A web-based tourism employment accelerator for youth and women in Kicukiro District, Rwanda.

## Platform Pages

- **Metrics Dashboard** - Overview stats and success metrics
- **Youth Dashboard** - Progress tracking, bootcamp schedule, skills
- **Training** - Training modules and curriculum
- **Credential** - ThinkSphere verified credential/profile
- **Recruiter Dashboard** - Search and browse alumni talent
- **Competition** - ThinkSphere x BDF Pitch Competition
- **Admin Panel** - Platform administration and analytics

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the platform.

## Built With

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
